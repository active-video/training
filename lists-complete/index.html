This illustrates lists and the edge cases related to them where animations should still be smooth even when they do not contain a sufficient number of items to run out of their parent container viewport.
Let's define a list that animates in a carousel manner to be made up of M items visible at a time. So in this case, M=5. So if any list has at least 5+2=7 items, then we are good.
